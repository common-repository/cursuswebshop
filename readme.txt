A plugin for connecting moodle to wordpress

== Description ==

Selling your courses in moodle with wordpress.

= Features =
* Import from moodle XML


== Installation ==

1. Upload plugin folder to the `/wp-content/plugins` directory
1. Activate the plugin through the 'Plugins' menu in WordPress

== Frequently Asked Questions ==

= How can I add webshop module into my page? =


= Where is the settings page for Timeline Calendar? =

Log into WordPress and go to Settings\Cursuswebshop.
